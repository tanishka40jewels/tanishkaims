🖼️ Bulk Image Upload & Auto-Tag
Upload multiple product images — the system matches each image to a product automatically
📌 Image Naming Rules (Auto-matching logic)
Images are matched to products using these rules in order:
1️⃣ Exact product code — name your file RH-KND-001.jpg → matches product code RH-KND-001
2️⃣ Partial code match — RH-KND-001_front.jpg → still matches RH-KND-001
3️⃣ Product name keywords — kundan_rani_haar.jpg → matches "Kundan Rani Haar Set" by keyword
4️⃣ Unmatched → shown as unmatched; you can manually assign below
